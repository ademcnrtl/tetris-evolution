# 🕹️ Tetris: The Evolution (1984 - 1989)

A faithful recreation and digital tribute to the genius of **Alexey Pajitnov**. This project explores the historical journey of the world's most famous puzzle game, meticulously recreating the visual aesthetics, hardware limitations, and unique soundscapes of its three most iconic eras.


## 📜 The Inspiration
Inspired by the **Tetris (2023) movie** and the incredible true story of its creation, this project aims to honor the memory of how a simple program on a Soviet **Elektronika 60** computer transcended borders to become a global phenomenon.


## 📺 Authentic Historical Themes

This version features three distinct modes, each designed to replicate the original hardware experience:

### 1. Elektronika 60 (1984) - "The Roots"
* **Visuals:** Authentic character-based (ASCII) graphics.
* **Special Effect:** Integrated **CRT Monitor Filter** with scanlines, screen curvature, and phosphor ghosting to mimic 80s Soviet hardware.
* **Vibe:** The raw, mechanical origin of the legend.

### 2. MS-DOS Version (1986) - "Western Migration"
* **Visuals:** The classic 16-color PC palette that introduced Tetris to the Western world.
* **Aesthetics:** Boxed block designs and clean UI representing early personal computing.
* **Logic:** Mimics the original PC keyboard latency and gravity mechanics.

### 3. GameBoy Edition (1989) - "The Global Legend"
* **Visuals:** The iconic 4-shade "pea-soup" green monochrome palette.
* **Sound:** Features the mystical **"Korobeiniki"** (Theme A) in its original 8-bit chiptune glory.
* **Immersion:** Recreated sound effects for line clears and rotations that defined handheld gaming history.

## 🛠️ Technical Highlights
* **Advanced Shaders:** Custom GLSL shaders used to simulate the "warmth," flicker, and distortion of vintage CRT screens.
* **Audio Engineering:** Frequency-limited audio engine to accurately mimic the GameBoy's LR35902 sound chip.
* **Authentic Logic:** The rotation systems and gravity steps are tailored to match each specific era's original source code behavior.
